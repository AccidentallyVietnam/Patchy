Patchy for Windows

An open-source bittorrent client with all of the features and none of the ads

Regular Expressions

Patchy supports adding torrents based on regular expressions in RSS feeds. These are powerful expressions that allow you to be quite specific with which torrents are added and which are discarded. It is suggested that you spend some time reading about them on Wikipedia, or through other sources, but here's a really simple crash course:

Say you want to add new episodes of the TV show "foo bar" as they're released. You know the torrent name will be something like "Foo Bar S01E#". The following regex would match it:

Foo Bar S01E[0-9]+

This looks for "Foo Bar SE01E", and then at least one character between 0 and 9 ([0-9]+).

Another example: you want the latest album by "That One Group". You know the torrent will be named something like "That One Group - Album Name". You could use this regex:

That One Group - [A-Z ]+

This will match on "That One Group - ", and then at least one letter or space ([A-Z ]+).

If you're in doubt, regex will also match most normal text. So, "Foo Bar" would match anything with "Foo Bar" in the title. There are a few special characters you cannot use - see the Wikipedia page. You can still use these characters if you escape them, like this: "\."

Notes:

  • Patch regexes are not case-sensitive
  • ^ and $ are not added automatically
